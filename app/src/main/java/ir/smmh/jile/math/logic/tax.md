# $t(x)$

| $x$                | $t(x)$ | $\min(x\cdot t(x))$ | $\max(x\cdot t(x))$ |
| ------------------ | ------ | ------------------- | ------------------- |
| $x < 10$           | 24%    | 0                   | 2.4                 |
| $10 \le x < 100$   | 12%    | 1.2                 | 12                  |
| $100 \le x < 1000$ | 4%     | 4                   | 40                  |
| $1000 \le x$       | 1%     | 10                  | $\infin$            |
